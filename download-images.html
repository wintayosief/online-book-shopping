BookHaven - Download Book Cover Images

How to Download Images:

  1. Right-click on each book cover image below
  2. Select "Save Image As..." or "Download Image"
  3. Save each image to your "images" folder with the corresponding filename
  4. Alternatively, click the "Download" button next to each image

Note: After downloading, you can update the image paths in js/main.js to use local paths like "images/gatsby.jpg" instead of the full URLs.
